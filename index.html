python-twitch 1.2¶

Warning

This documentation is currently a work in progress and is still rough around the edges. Not finding what you are looking for? Please open an issue describing your troubles on python-twitch’s github page!

python-twitch is a library designed for easy access to the twitch api’s. It is written in Python and enables you to access most of the functionality of the official twitch-api documentation while also offering an interface towards their hidden api and usher services.

Features¶

python-twitch 1.2 currently supports:

  • access to twitch api’s
    • v3 api
    • hidden api
    • usher api

  • translation of m3u8 playlists into more friendly json format

  • tests!
